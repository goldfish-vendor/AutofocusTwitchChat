# Auto-Focus Chat for Twitch

Automatically focuses the Twitch chat input when clicking outside the video player, so you can start typing instantly.

## Features

- Click anywhere and start typing in chat
- Hold Ctrl/Cmd to avoid focusing chat
- Does not interfere with Twitch video shortcuts
- Lightweight and simple

## Installation

Install from the Chrome Web Store.

You may also download the zip here and side-load it with developer mode.
